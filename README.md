# 🚀 Proyecto Base: Prácticas de Git & GitHub

Bienvenido al repositorio de entrenamiento. Este proyecto web simula un entorno corporativo con múltiples recursos interconectados.

## 📂 Arquitectura
- `/assets`: Recursos estáticos (CSS, JS, SVG).
- `/pages`: Vistas HTML.
- `/data`: Archivos de configuración (JSON).
- `/docs`: Documentación interna del equipo.

## 🎯 Objetivo
Utiliza este repositorio para practicar:
1. Creación y fusión de ramas (`git merge`).
2. Resolución de conflictos en archivos como `data/settings.json`.
3. Trabajo colaborativo mediante Pull Requests.
